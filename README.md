# NTask API

JavaScript API for task management

## Pre-requisites

- node 17x

## Modules

- Express
- Sequelize
- Passport
- Mocha
- Chai
- ApiDoc

## Installation

- Install dependencies

```bash
npm install
```

- Create .env

```bash
cp .env.example .env
```

- Start server

```bash
npm start
```

- Run tests

```bash
npm test
```
